    mixins

    mq

    @mixin mq($key) { ... }

    Description

    A mixin for media queries.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $key

    A key from the breakpoints sass map.

    		Unquoted text— none

    Example

    usage

    @include mixins.mq(small){
  .body { 
      font-size: 2rem;                
      }        
}

    result

      @media only screen and (min-width: 40em) {
    .body { font-size: 2rem; }
}

    Requires

    • [variable] breakpoints
    • [external] bp.$breakpoints — The breakpoints sass map. Should be defined in '_breakpoints.scss'

    pseudo

    @mixin pseudo($loc: before, $content: '', $pos: absolute, $top: 0, $bottom: 0, $left: 0, $right: 0) { ... }

    Description

    A positioning pseudo element.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $loc

    The pseudo element. Either before OR after.

    		Unquoted textbefore
    $content

    the content of the element.

    		Unquoted text''
    $pos

    the position.

    		Unquoted textabsolute
    $top

    The top constraint of the object.

    		Number0
    $bottom

    The bottom constraint of the object.

    		Number0
    $left

    The left constraint of the object.

    		Number0
    $right

    The right constraint of the object.

    		Number0

    Example

    positioned pseudo usage

    .card {
  @include mixins.pseudo($loc:before, $pos:relative, $top:20);
}

    positioned pseudo result

    .card {
  position: relative;
}
.card::before {
  content: "";
  position: relative;
  top: 20;
  bottom: 0;
  left: 0;
  right: 0;
}

    Output

    'positioned pseudo element'
